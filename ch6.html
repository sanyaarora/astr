Time Series Analysis (Chapter 11)

In this section of the course, we covered an overview of statistical techniques used in time serires analysis. Time series is useful when we have time-varying data, such as a pulsar with periodically varying luminosity. The main topics we discussed are:

Smoothing: Smoothing techniques are used to reduce noise and clarify signals in time series data. They help in highlighting underlying trends by averaging out the variations. We can use methods like central moving average (CMA) or exponential weighted moving average (EWMA).

Autocorrelation Functions (ACF): Measures the correlation between a time series and lagged versions of itself. It quantifies how much current values of the series are related to past values. ACF is used to identify repeating patterns within a time series. Partial Autocorrelation Functions (PACF) is an alternative to ACF that isolates the direct effect of past observations on the current observation, excluding indirect correlations from intermediate lags. Some models for generating ACFs are as follows:

Random Walk Model: Assumes that the future value of a variable is a random step from its current value, useful in predicted random data.
Autoregressive (AR) Model: Allows predicted values to be dependant on past values with different weights.
Moving Average (MA) Model: Instead of depending on past values of the variable, predicted values depend on past values of noise.
ARMA and ARIMA Models: ARMA combines both autoregressive and moving average models to analyze stationary time series, while ARIMA includes differencing of raw observations to make the time series stationary before applying ARMA.
ARCH Model: Allows variance to change over time, which is ideal for data without a fixed variance.

Discrete Correlation Functions (DCF): Useful when we have unevenly sampled data. Computes the ACF without assuming even spacing of the data. Can be used to find the ACF of one data set or to find cross-correlations between two datasets.

Structure Function: Investigates the variability of a time series, providing the temporal correlation structure without needing the data to be stationary.

Fourier Power Spectrum and Periodograms: Evaluates the intensity of periodic signals or frequencies present in data, transforming data from the time domain to the frequency domain. This helps us identify cyclic trends in data. The Lombâ€“Scargle periodogram is a commonly used method that allows for unevenly spaced data.

ANOVA: Used to determine if there are any statistically significant differences between the means of multiple independent groups.

These techniques are widely used to identify time-dependant structures in data. They are commonly used in Astronomy to identify periodic stars.
