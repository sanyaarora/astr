Multivariate Analysis (Chapter 8)

During this part of the course, we explored multivariate analysis, which involves interpreting datasets with several variables. The goal is to find the distribution of data in a multidimensional space and reduce the number of variables to simplify the dataset's structure. We discussed several statistical methods for multivariate analysis:

Hotelling’s T² statistic: This is an extention of the t-test to the multivariate space. It tests whether the means of populations are significantly different when data are multidimensional.

Multiple Linear Regression: A statistical method that models the relationship between 2 or more variables x_i and a response variable y by fitting a linear equation to the data. The regression line is calculated by finding the minimum sum of squared deviations of the predicted values from the actual values.

Principal Component Analysis (PCA): This method reduces dimensionality by transforming the original variables into a new set of variables (principal components), which are uncorrelated. The first few components contain most of the variance of the dataset. We can then analyze the principal components to determine which variables are the most related to each other.

Factor Analysis: Used to describe variability among observed, correlated variables in terms of a lower number of variables called factors. This is useful in situations where we are identifying underlying variables, or factors, that explain a pattern of correlations in observed variables.

Overall, multivariate analysis has many uses across different fields. One example could be in ecological studies, such as analyzing environmental impacts on coral reef systems. We could use PCA to reduce the dimensions of environmental data (such as water temperature, salinity, nutrient concentrations, light levels, etc) to identify the main factors affecting coral health.
