Probability Concepts (Chapters 2, 3, and 4)

In this part of the course, we learned about various probability concepts, probability distributions, and hypothesis testing. Probability is a measurement of how likely an event is to occur within a set of outcomes. It's useful when assessing risk and making predictions. Probability ranges from 0 (an event will not occur) to 1 (an event will definitely to occur).

A probability distribution describes the probabilities of occurence of different outcomes of a random variable. These distributions can be discrete or continuous. Some common probability distribution functions are:

Binomial Distribution: Describes the number of successes in a fixed number of independent trials, with each trial having the same probability of success. For example, flipping a coin multiple times.
Poisson Distribution: Models the number of times an event occures in a fixed time interval, assuming the events occur with a known mean and are independent of time.
Normal (or Gaussian) Distribution: A distribution shaped like a bell-shaped curve. The normal distribution is symmetric, and the width of the curve is the standard deviation of the dataset, and the max of the curve is the mean.

Confidence intervals provide a range of values which contain the true value of a parameter with a certain probability. This is important for being able to tell the accuracy of statistical estimates. For example, if we measure the mean height of a sample of trees and calculate a 95% confidence interval, we can be 95% confident that the interval contains the true mean height.

Hypothesis testing is used to estimate the validity of a claim. It starts with a null hypothesis that represents a default position. We then use sample data to determine if we can reject the null hypothesis. Some common hypohtesis tests are:

Z-test: Checks whether the means of two datasets are different. Used when we know the population variance and the sample size is large.
T-test: Compares the means of two datasets. Used when the population variance is unknown and the sample size is small.
